# TRIBE JUMP
Inspired by the people of masai tribe , this is a a game made in GODOT engine for the Resolution WEEK 3 of game dev path, game gets hard as you jump and go up, you cando tripple jump, Insane jump boosts

## controls:
* a and d = left and right
* space or enter = jump
